Task 3: Paragraph on CSS Selectors

CSS Selectors are patterns used to select elements for styling in HTML documents. They allow developers to apply styles based on the attributes of elements, such as their id, class, or type. Here's an overview of key CSS selectors:

  • ID Selector: The ID selector selects an element with a specific `id` attribute. The ID value must be unique within a page.
    Example: #firstName { color: blue; } will target the element with the id `firstName` and apply a blue color.
  • Class Selector: The class selector selects all elements with a specific class. Multiple elements can share the same class.
    Example: .form-container { background-color: yellow; } will apply a yellow background color to all elements with the class `form-container`..

Difference Between ID and Class Selectors: An ID selector targets a unique element on the page, while a class selector can be used to target multiple elements. The ID selector has higher specificity than the class selector and is ideal for targeting single elements, whereas class selectors are better for applying styles to multiple elements.

Task 4: Paragraph on CSS Box Model

The CSS Box Model represents the structure of an element's content box and its surrounding padding, border, and margin. It is essential for controlling the layout and spacing of elements on a web page. The Box Model includes the following components:

  • Padding: Padding is the space between the content of an element and its border. It is used to create space inside the element.
  • Border: The border is the line that wraps around the padding (if any) and the content. It can be styled in various ways (width, color, and style).
  • Margin: The margin is the space outside the element, separating it from other elements on the page.

The Box Model allows you to control the layout of elements by adjusting the size and space around them. Understanding the Box Model is crucial for effective web design.
